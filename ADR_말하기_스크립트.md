“AI 자기소개서 첨삭 앱을 Claude API와 SSE 스트리밍 방식으로 구현하기로 결정했습니다.

처음에는 일반 REST 방식과 저렴한 모델도 고려했지만, 첨삭 응답이 길어 사용자가 오래 기다려야 한다는 문제가 있었습니다.

그래서 결과를 실시간으로 보여줄 수 있는 SSE 스트리밍 방식을 선택했고, 첨삭 품질을 위해 Claude Opus 모델을 사용했습니다.

비용 문제는 Prompt Caching으로 보완했습니다.

결국 저희는 사용자 경험과 첨삭 품질을 가장 중요한 기준으로 판단해 이 구조를 선택했습니다.”


Prompt Caching : 프롬프트 앞부분(Prefix)의 연산 결과를 저장해 두었다가 재사용하여 비용과 지연 시간(Latency)을 혁신적으로 줄이는 기술
