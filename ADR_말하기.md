
### 1. 우리는 무엇을 선택했나?
AI 자기소개서 첨삭 앱

### 2. 다른 선택지는 무엇이었나?
과제 요약 햅, AI 일정 생성 앱

### 3. 우리 팀의 기준에서 왜 이 선택이 더 낫나?

* 시간·비용 부담 없이 즉각적인 피드백 가능
* 자기소개서 완성도를 빠르게 향상 가능
* 반복 수정 과정의 어려움 감소
* 언제 어디서든 쉽게 사용 가능
* 기존 첨삭 서비스보다 접근성과 효율성이 높음

### Claude API + SSE 스트리밍 을 왜 선택 했는가?
Claude API로 AI 첨삭 기능을 만들고, SSE 스트리밍으로 답변을 실시간 출력해서 사용자가 기다리지 않고 바로 결과를 읽을 수 있기에 효율 성이 좋기 떄문에
