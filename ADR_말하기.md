
--

## 1. 우리는 무엇을 선택했나?

**Claude API + SSE 스트리밍** 방식으로 AI 자기소개서 첨삭 앱을 구현

---

## 2. 다른 선택지는 무엇이었나?

| 선택지 | 내용 |
|---|---|
| 일반 REST 방식 | 응답을 한 번에 반환 — 사용자가 결과를 기다려야 함 |
| Sonnet / Haiku 모델 | 비용이 저렴하지만 첨삭 품질이 낮음 |

---

## 3. 우리 팀의 기준에서 왜 이 선택이 더 낫나?

- 시간·비용 부담 없이 **즉각적인 피드백** 가능
- 자기소개서 완성도를 **빠르게 향상** 가능
- 반복 수정 과정의 어려움 감소
- 언제 어디서든 **쉽게 사용** 가능
- 기존 첨삭 서비스보다 **접근성과 효율성**이 높음

---

## 4. Claude API + SSE 스트리밍을 선택한 핵심 이유

> Claude API로 고품질 AI 첨삭 기능을 구현하고,
> SSE 스트리밍으로 답변을 **실시간 출력**하여
> 사용자가 기다리지 않고 바로 결과를 읽을 수 있어 **UX 효율성**이 높습니다.
