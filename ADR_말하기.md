
## ADR 60초 말하기 카드

> ⏱️ **60초** 말로 설명

### 1. 우리는 무엇을 선택했나?
Claude API + SSE 스트리밍 방식으로 AI 자기소개서 첨삭 앱을 만들었습니다.

### 2. 다른 선택지는 무엇이었나?
응답을 한 번에 반환하는 일반 REST 방식도 있었고, 더 저렴한 Sonnet/Haiku 모델도 고려했습니다.

### 3. 우리 팀의 기준에서 왜 이 선택이 더 낫나?
첨삭 결과가 길기 때문에 스트리밍으로 실시간 출력해야 UX가 좋습니다. Opus 모델은 품질이 높고, prompt caching으로 비용도 줄였습니다.
