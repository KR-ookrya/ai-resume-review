# 비전 & 목표 — AI 자기소개서 첨삭 서비스

## 한 줄 비전

> **"모든 취업 준비생이 전문 컨설턴트 수준의 첨삭을 무료로 받을 수 있는 세상"**

---

## 왜 이 서비스인가 (Problem Statement)

자기소개서는 취업의 첫 관문이지만, 대부분의 취준생은:

- 전문 첨삭 비용(1건당 3만~10만 원)을 감당하기 어렵다.
- 주변에 현직자나 경력자에게 솔직한 피드백을 구하기 쉽지 않다.
- 혼자 수십 번 고쳐도 어디가 문제인지 객관적으로 파악하기 힘들다.

Claude Opus 4.7의 언어 이해 능력을 활용하면, 10년 경력 컨설턴트 수준의 심층 피드백을 **즉시·무료·무제한**으로 제공할 수 있다.

---

## 핵심 가치

| 가치 | 의미 |
|------|------|
| **즉각성** | 입력 즉시 스트리밍으로 결과를 실시간 확인 |
| **구체성** | 점수·항목별 근거·수정 예시까지 포함한 실질적 피드백 |
| **접근성** | 브라우저만 있으면 누구나 — 회원가입·설치 불필요 |
| **신뢰성** | Claude Opus 4.7 기반, 직무·회사 맥락을 반영한 맞춤 분석 |

## 기술 스택 & 설계 원칙

```
Frontend  : Vanilla HTML / CSS / JS  — 의존성 최소화, 빠른 로드
Backend   : Node.js + Express        — 경량 서버, SSE 스트리밍
AI        : Anthropic SDK (claude-opus-4-7) — 최고 품질 한국어 이해
```

**설계 원칙**

1. **No over-engineering** — 기능 하나 추가할 때 기존 구조를 깨지 않는다.
2. **Streaming first** — 사용자가 기다리지 않도록, 생성 즉시 화면에 출력한다.
3. **Korean-native** — 프롬프트·UI 모두 한국 취업 시장에 최적화된다.
